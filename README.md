SpreeStreamline
=============


Requirements
============

* Spree Core 2.0.0 or greater.
* Ruby 1.9.2 or greater.

Installation
============

1. Add `gem 'spree_streamline', github: 'jtapia/spree_streamline'` to Gemfile
1. Run `bundle`

Testing
=======

1. bundle exec rake test_app
1. bundle exec rspec spec
